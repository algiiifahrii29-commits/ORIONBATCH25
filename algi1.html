1. APA ITU VERSION CONTROL?

Version Control adalah sistem yang digunakan untuk menyimpan dan mengatur perubahan pada sebuah file atau kode program dari waktu ke waktu. Dengan version control, kita bisa: Melihat riwayat perubahan kode Mengembalikan kode ke versi sebelumnya jika terjadi kesalahan Bekerja bersama tim tanpa takut kode saling tertimpa Contoh sederhana Misalnya kamu membuat program hari ini: Versi 1 → Program awal Versi 2 → Menambahkan fitur login Versi 3 → Memperbaiki bug Semua perubahan tersebut tersimpan rapi, jadi jika versi terbaru rusak, kamu bisa kembali ke versi sebelumnya. Singkatnya: Version Control = sistem untuk mengelola dan menyimpan riwayat perubahan kode program.

2. PERBEDAAN GIT DAN GITHUB? ?

Git adalah software atau tools yang digunakan untuk mengelola version control pada komputer kita. Fungsi Git: Menyimpan riwayat perubahan kode Mengatur versi program Menggabungkan kode dari beberapa programmer Git biasanya digunakan melalui terminal atau command line. sedangkan GitHub adalah website atau layanan online untuk menyimpan repository Git di internet. Fungsi GitHub: Menyimpan project secara online Memudahkan kerja sama tim Menjadi backup project Bisa melihat dan mengelola kode melalui website

3. PENJELASAN APA ITU HTML?

HTML (HyperText Markup Language) adalah bahasa dasar yang digunakan untuk membuat struktur halaman website. HTML digunakan untuk menentukan: Judul halaman Paragraf Gambar Link Tabel Form HTML bukan bahasa pemrograman, tetapi bahasa markup yang digunakan untuk menyusun tampilan dasar website.
